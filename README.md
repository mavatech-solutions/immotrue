# ImmoTrue

KI-gestützte Immobilienanalyse für den DACH-Raum (Deutschland, Österreich, Schweiz).

## Struktur

- `mobile/` — React Native + Expo App (iOS/Android)
- `web/` — Astro.js Landing Page + Web-App
- `admin/` — Astro.js Admin Dashboard
- `backend/` — Supabase (Datenbank, Auth, Edge Functions, Cron Jobs)
- `shared/` — Geteilte TypeScript Types & Utilities
- `docs/` — Projektdokumentation

Siehe [ImmoTrue-Master-Plan.md](./ImmoTrue-Master-Plan.md) für den vollständigen Bauplan.
